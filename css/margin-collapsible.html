    margin 的传递和折叠问题

    # 什么是 margin 的传递

    举个例子: 父子两个盒子, 要求子元素距离父元素左上 50px 的间距.

    实现代码

    <style>
  .super-box {
    width: 200px;
    height: 200px;
    background-color: red;
  }

  .child-box {
    width: 100px;
    height: 100px;
    background-color: yellow;
    margin-left: 50px;
    margin-top: 50px;
  }
</style>
<div class="super-box">
  <div class="child-box"></div>
</div>

    上面的代码给子元素设置了 margin , margin-left 达到了预期的效果但是 margin-top 效果非常怪异: 父子元素顶部重叠,并且一起向下偏移了. 好像子元素设置的 margin-top 作用在了父元素的身上. 这就是 margin-top 的传递

    # margin 传递的特点

    • 在父子元素之间
    • margin 左右不会传递, 只会上下传递

    # margin-top 传递

    • 块级元素的顶部线和父元素顶部线重叠
    • 块级元素的 margin-top 传递给父元素

    # margin-bottom 传递

    • 块级元素底部和父元素底部线重叠
    • 父元素高度 auto
    • 块级元素的 margin-bottom 传递给父元素
    .super-box {
  width: 200px;
  height: auto;
  /* height: 200px; */
  background-color: red;
}

.child-box {
  width: 100px;
  height: 100px;
  background-color: yellow;
  margin-left: 50px;
  margin-bottom: 50px;
}

    # 如何解决 margin 传递问题

    # 1.父子关系直接的距离最好使用 padding 来设置

    .super-box {
  width: 200px;
  height: 200px;
  background-color: red;
}

.child-box {
  width: 100px;
  height: 100px;
  background-color: yellow;
}

    # 2.给父元素设置一个透明的 bodrder

    border: 1px solid transparent;

    这种方式不太好,虽然 border 看不见,但是他仍然占据空间

    .super-box {
  width: 200px;
  height: 200px;
  background-color: red;

  border: 1px solid transparent;
}

.child-box {
  width: 100px;
  height: 100px;
  background-color: yellow;
  margin-top: 50px;
  margin-left: 50px;
}

    # 3.触发 BFC

    父元素设置 overflow: auto;

    .super-box {
  width: 200px;
  height: 200px;
  background-color: red;

  overflow: auto;
}

.child-box {
  width: 100px;
  height: 100px;
  background-color: yellow;
  margin-top: 50px;
  margin-left: 50px;
}

    触发 BFC 使得盒子占据独立的空间不受外界影响

    # margin 和 padding 使用区别

    如何设置一个父子元素,子元素距离父元素的左上角有一定的间距?

    其实 CSS 里有很多种实现的方案, 例如可以设置父级元素的内边距, 也可以设置子元素的 margin 都可以实现上面的效果.

    个人更加倾向使用 pading . 因为从语义的角度来说 margin 更加表示了同一个级别元素之间的距离, 而 padding 表示了内部的关系.

    注意使用 padding 的时候会把盒子撑大, 可以使用 box-sizing 来解决这个问题

    padding: 10px;
box-sizing: border-box;

    # 上下 margin 的折叠

    垂直方向上相邻的 2 个 margin(margin-top, margin-bottom) 有可能会合并成一个 margin, 这种现象叫 collapse (折叠).

    <style>
  .box-1 {
    height: 100px;
    background-color: red;
    margin-bottom: 20px;
  }
  .box-2 {
    height: 100px;
    background-color: yellow;
    margin-top: 40px;
  }
</style>
<div class="box-1"></div>
<div class="box-2"></div>

    水平方向永远不会折叠

    折叠后最终值的计算规则

    • 两个值比较取大值

    如何防止 margin 折叠

    • 只设置一个元素的 margin
