    # 巧妙使用 margin 负值

    # 题目

    实现固定容器宽度 1040, 宽度刚好容纳 5 个宽度为 200px 的 item 并且 item 之间的间距是 10px

    <style>
  .content {
    width: 1040px;
    height: 610px;
    background-color: yellowgreen;
  }
  .item {
    width: 200px;
    height: 300px;
    background-color: skyblue;
  }
</style>
<div class="content">
  <div class="item item0">0</div>
  <div class="item item1">1</div>
  <div class="item item2">2</div>
  <div class="item item3">3</div>
  <div class="item item4">4</div>
  <div class="item item5">5</div>
  <div class="item item6">6</div>
  <div class="item item7">7</div>
  <div class="item item8">8</div>
  <div class="item item9">9</div>
</div>

    # 解决方案

    实现这个需求有很多的方案, 比如使用 flex 布局

    # 1.flex

    <style>
  .content {
    width: 1040px;
    height: 610px;
    background-color: yellowgreen;

    display: flex;
    justify-content: space-between;
    flex-wrap: wrap;
    align-content: space-between;
  }
  .item {
    width: 200px;
    height: 300px;
    background-color: skyblue;
  }
 
</style>
<div class="content">
  <div class="item item0">0</div>
  <div class="item item1">1</div>
  <div class="item item2">2</div>
  <div class="item item3">3</div>
  <div class="item item4">4</div>
  <div class="item item5">5</div>
  <div class="item item6">6</div>
  <div class="item item7">7</div>
  <div class="item item8">8</div>
  <div class="item item9">9</div>
</div>

    # 2.浮动

    如果对兼容性要求很高那么可以使用浮动, 浮动的兼容性比 flex 还要好早期的 IE 都支持

    .item {
  width: 200px;
  height: 300px;
  background-color: skyblue;

  margin-right: 10px;
  float: left;
}

    给 item 设置浮动并且右边距设为 10, 这时候还需要解决每一行最后一个元素排不下的问题.

    .item:nth-child(5n) {
  margin-right: 0px;
}

    使用 nth-child 能够解决这个问题, 但是 nth-child 的兼容性不好所以采用.

    接下来就是重点了: 巧妙使用 margin 负值来解决

    块级内容宽度计算公式:

    父级块元素的宽 = 子元素内容 + margin-left + margin-right + border + padding

    一行只能放 4 个的根本问题是容器的宽度不够(差 10px).

    实际上子元素内容占用宽度 = 5 * (200(item 宽度) + 10px(间距)) = 1050px, 而父级容器宽度只有 1040px.由于 margin, border, padding 这些值都是 0, 子元素内容是 1050px 计算结果就是 1050px,导致差了 10px. 那么可以把 margin 设置成 -10px 那么就正好可以放下了.

    <style>
  .content {
    width: 1040px;
    height: 610px;
    background-color: yellowgreen;
  }
  .box {
    margin-right: -10px;
  }
  .item {
    width: 200px;
    height: 300px;
    background-color: skyblue;
    margin-right: 10px;
    margin-bottom: 10px;
    float: left;
  }
</style>
<div class="content">
  <div class="box">
    <div class="item item0">0</div>
    <div class="item item1">1</div>
    <div class="item item2">2</div>
    <div class="item item3">3</div>
    <div class="item item4">4</div>
    <div class="item item5">5</div>
    <div class="item item6">6</div>
    <div class="item item7">7</div>
    <div class="item item8">8</div>
    <div class="item item9">9</div>
  </div>
</div>

    # 多列布局边框问题

    一个固定宽度的容器放着多个固定宽度的子元素,要求像九宫格那种带边框

    <style>
  .content-border {
    width: 1010px;
    height: 400px;
    background-color: #eee;
  }

  .content-border .item {
    width: 200px;
    height: 200px;
    border: 2px solid black;

    background-color: white;
    margin-left: -1px;
    margin-right: -1px;
    float: left;
  }
</style>
<div class="content-border">
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
</div>

    解决步骤:

    • 子元素浮动, 给子元素设置边框.
    • 两个紧挨的 item 的中间边框位置宽度就会变成 2 倍边框宽度,给 margin 设置负值消除一个宽度(可以设置 margin: -边框宽度,也可以 margin-left margin-right 各一半)

    最终还是会多出来一个边框的宽度.可以给某一个 item 的宽度减少一个边框宽度

    <style>
  .content-border {
    width: 1100px;
    height: 800px;
    margin: 0 auto;
    background-color: #eee;
  }

  .content-border .box {
    margin-left: 1px;
  }
  .content-border .item {
    width: 221px;
    height: 168px;
    border: 1px solid black;

    background-color: white;
    margin-left: -1px;
    box-sizing: border-box;
    float: left;
  }
  .content-border .item.first {
    width: 220px;
  }
</style>
<div class="content-border">
  <div class="box">
    <div class="item first"></div>
    <div class="item"></div>
    <div class="item"></div>
    <div class="item"></div>
    <div class="item"></div>
  </div>
</div>
