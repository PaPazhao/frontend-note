    CSS 的定位

    # 标准流

    默认网页的文档安装标准流进行布局的, 他具备下面特点:

    • 块级元素独占一行
    • 多个行内元素共站一行
    • 元素和元素默认不会层叠
    • 布局过程是把元素按照从上到下,从左到右依次排列

    # 标准流元素调整位置

    在标准流里面使用 margin(可以设置负值), padding 对元素进行调整位置.

    # 缺点:

    • 设置一个元素的 margin 或者 padding 通常会影响到标准流中其他元素的定位效果
    • 不能实现元素层叠的效果

    如果我们希望一个元素可以跳出标准流单独对某个元素定位, 那么就需要通过设置 position 属性来对元素定位

    # 元素定位

    元素定位就是把元素从标准流里面取出来单独对他进行定位. 元素定位可以实现下面的效果:

    • 实现元素层叠,一个元素可以放在另一个元素的上面
    • 固定在视口的某一个位置,例如导航栏始终在顶部的位置

    实现元素定位则需要使用 position 属性

    # position 属性

    position 属性可以对元素进行定位, 常用的值有 5 个

    • static 默认值,静态定位(标准流)

    • relative: 相对定位

    • absolute: 绝对定位

    • sticky: 固定定位

    • fixed: 粘性定位

    除去 static 其他的值可以让元素变成定位元素

    # 静态定位

    position 属性的默认值 static:

    • 元素按照标准流进行布局
    • 标准流里设置 left, right, top, bottom 没有任何作用. 定位元素才有效

    # 相对定位

    • 元素依然按照标准流进行布局
    • 可以通过 left, right, top, bottom 对元素位置调整, 元素位置的调整不影响其他元素定位
    • left, right, top, bottom 的值是以元素原来的位置作为参考点的

    # 相对定位使用场景

    一般用来微调, 不影响其他元素位置的前提下对当前元素位置进行调整

    # 固定定位 fixed

    # 特点

    • 元素脱离标准流
    • 可以使用 left, right, top, bottom 进行定位
    • 定位参照物是视口

    # 固定定位使用场景

    • 返回顶部的按钮
    • 固定的工具条

    视口: 浏览器文档的可是区域

    画布: 渲染文档的区域

    # 绝对定位 absolute

    # 绝对定位的特点

    • 元素脱离标准流
    • 可以通过 left, right, top, bottom 进行定位
    • 定位参照物是最邻近的祖先并且是定位元素
    • 如果没有样的祖先元素那就是是视口
    .box-container {
  width: 1000px;
  height: 1000px;
  background-color: #f00;
  position: relative;
}
.box {
  width: 500px;
  height: 500px;
  background-color: #0f0;
  position: absolute;
  top: 0;
  right: 0;
}

    # 子绝父相

    绝大多数情况下子元素的绝对定位都是相对于父元素进行定位

    如果希望子元素相对于父元素进行定位,同时又不希望父元素脱标,常用的解决方案是:

    • 父元素设置相对定位(让父元素成定位元素而且不脱标)
    • 子元素设置绝对定位

    # 绝对定位元素的特点(fixed/absolute)

    当元素定位属性是 absolute 和 fixed 的时候,这个元素就是绝对定位元素.

    他们都是相对于一个参考对象进行定位

    脱离标准流 定位元素 绝对定位元素 定位参照对象
    static-静态定位 否 否 否 否
    relative-相对定位 否 是 否 元素原来的位置
    absolute-绝对定位 是 是 是 最邻近的祖先定位元素或者视口
    fixed-固定定位 是 是 是 视口

    绝对定位元素由于脱离标准流,不再关心你是块级元素还是行内元素

    • 随意设置宽高
    • 宽高默认由内容决定
    • 不受标准流的约束
    • 不再区分块级,行内级元素,块级行内级元素很多特性都会不起作用
    • 不再给父级元素汇报高度
    • 脱标元素内部默认还是按照标准流布局

    # 绝对定位元素的宽高计算公式

    • 定位参照对象的宽度 = left + right + margin-left + margin-right + 绝对定位元素事件占用宽度

    • 定位参数对象的高度 = top + bottom + margin-top + margin-bottom + 绝对定位元素的实际占用高度

    根据这个公式我们可以得出下面的

    1. 如果希望绝对定位元素的宽高和参照对象一样可以设置: left: 0; right: 0; top: 0; bottom: 0; margin: 0;
    2. 如果希望绝对定位元素在定位参照对象中居中显示,可以给绝对定位元素设置一下属性: left: 0; right: 0; top: 0; bottom: 0; margin: auto; 还得设置具体的宽高值

    # 粘性定位 sticky

    sticky 可以看做是相对定位和固定(绝对)定位的结合体, 他允许被定位的元素表现的像相对定位一样直到他滚动到某个阈值点然后就变成绝对定位.

    常见的使用场景就是手机通讯录列表按照拼音首字母分组,分组的标题滚动到顶部位置的时候就固定到顶部了.

    • top 是黏住的位置, 他是在最邻近祖先滚动试图的位置
    <style>
  .sticky-position .nav {
    position: sticky;
    top: 0;
    height: 44px;
    background-color: #f5f5f5;
  }

  .sticky-position li {
    height: 44px;
    background-color: #f9f9f9;
  }
</style>
<div class="sticky-position">
  <div class="nav">
    <span>A</span>
  </div>
  <ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
    <li>6</li>
    <li>7</li>
    <li>8</li>
    <li>9</li>
    <li>10</li>
  </ul>
</div>

    # z-index

    z-index 用来设置定位元素的层叠顺序, 取值范围是正整数, 负整数, 0

    比较原则: 兄弟关系

    • z-index 值越大,层叠越在上面
    • z-index 值相等,后面的元素层叠在上面

    非兄弟关系

    • 各自从自己以及祖先元素中找出最邻近的两个定位元素比较
    • 这两个定位元素必须有设置 z-index 具体数值
