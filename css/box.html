    CSS 盒子模型

    # 盒子模型的属性

    • content 内容(width/height)
    • padding 内边距
    • border 边框
    • margin 外边距

    盒子有四边,所以 padding / border / margin 也有四个边 top / bottom / left /right.

    # 内容

    内容是通过 width / height 来设置的.

    注意: 对于行内非替换元素设置 width / height 是无效的

    # width 的默认值

    width 默认值是 auto, 也就是说浏览器来决定 width. 虽然都是 auto 但是在不同的元素类型表现是不一样的

    • 跨级元素: 独占一行
    • 行内元素: 包裹内容

    # min-width / max-width

    另外还有下面的的属性来限制宽度的范围:

    • min-width: 最小宽度
    • max-width: 最大宽度

    移动端适配可以设置最大和最小宽度.

    和高度对应的还有 min-height / max-height 但是不常用

    # 内边距 padding

    内边距表示的是边框和内容之间的间距, 他有四个方向:

    • padding-top
    • padding-right
    • padding-bottom
    • padding-left

    也可以使用简写的方式

    .element {
  padding: top right bottom left;
}

    当四个方向的值有缺失的时候那么就使用对称位置的值补全.

    .element {
  padding: 10px 20px;
}

    等效于

    .element {
  padding: 10px 20px 10px 20px;
}

    # border

    border 用来设置边框, 他比其他属性特殊一点具备多个属性

    # border 的属性

    • width
    • style
    • color

    # border 的方向

    跟 pading 一样 border 也有四个方向, 他的属性也对应四个方向.

    以 width 为例子

    • border-top-width
    • border-right-width
    • border-bottom-width
    • border-left-width

    # style 的值

    • solid

    • dotted

    • dashed

    • double

    • groove

    • ridge

    • inset

    • outset

    • none

      • 和关键字 hidden 类似，不显示边框。在这种情况下，如果没有设定背景图片，border-width 计算后的值将是 0，即使先前已经指定过它的值。在单元格边框重叠情况下，none 值优先级最低，意味着如果存在其他的重叠边框，则会显示为那个边框。

    • hidden

      • 和关键字 none 类似，不显示边框。在这种情况下，如果没有设定背景图片，border-width 计算后的值将是 0，即使先前已经指定过它的值。在单元格边框重叠情况下，hidden 值优先级最高，意味着如果存在其他的重叠边框，边框不会显示。

    # 缩写

    border 也可以缩写, 顺序任意的

    .element {
  border: 2px dotted red;
}

    # margin

    外边距表示不同盒子之间的距离, 和 padding 一样 margin 也是有四个方向, 也可以进行简写.

    .element {
  margin-top: 10px;
  margin-bottom: 10px;
  margin-left: 20px;
  margin-right: 20px;
  /* 缩写 */
  margin: 10px 20px;
  margin: 10px 20px 10px 20px;
}

    # 外轮廓 outline

    outline 表示元素外轮廓, 语法和 border 一样

    • 不占用空间
    • 默认显示在 border 的外面
    outline: 1px solid red;

    a {
  /* 行内元素设置 margin-top 无效 */
  margin-top: 100px;
  display: inline-block;
  outline: none;
}

    # 阴影 box-shadow

    box-shadow 可以设置一个或者多个阴影

    • 每个阴影用<shadow> 表示
    • 多个阴影直接逗号隔开,从前到后叠加

    参数

    • offset-x 偏移
    • offset-y
    • blur-radius 模糊半径
    • spread-radius 延伸半径
    • color 阴影颜色,如果没有设置就跟随 color 的设置
    • inset 阴影是往外还是往里

    https://html-css-js.com/css/generator/box-shadow

    # 文字阴影 text-shadow

    • offset-x 偏移
    • offset-y
    • blur-radius 模糊半径
    • color 阴影颜色,如果没有设置就跟随 color 的设置
    • inset 阴影是往外还是往里

    https://html-css-js.com/css/generator/text-shadow/

    # box-sizing

    box-sizing 用来设置盒子模型中宽高行为

    • content-box 默认值, padding / border 都在 width, height 里面
    • border-box padding, border 都布置在 width, height 里面
