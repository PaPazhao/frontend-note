    CSS 的函数

    # var

    CSS 中可以自定义属性. 属性名以两个减号开头.属性值可以是任何有效的 CSS 值

    /* 定义变量 */
/* 只有后代元素可以用 */
/* 或者使用 :root */
html {
  --primary-color: #5f5f5f;
}

.box {
  /* 使用变量 */
  color: var(--primary-color);
}

    这样我们可以把视觉规范里面的颜色,字体通过变量来定义,这样当规范调整的时候直接修改变量值就好了.

    规则集定义的选择器是自定义属性的可见作用域(只在选择器内部有效)

    因此推荐将自定义属性定义在 html 中, 也可以使用 :root 选择器

    # calc

    计算 CSS 值, 通常用于计算元素大小或位置.

    计算支持加减乘除的运算, 运算符两边要有空白符.

    .item {
  /* with 的百分比是相对于包含块的(通常是父元素) */
  width: calc(100% - 100px);
}

    # blur

    blur 函数将高斯模糊应用与输出图片或者元素,通常和下面两个属性一起使用

    • filter 将模糊或者颜色偏移等图形效果应用于元素
    • backdrop 为元素后面的区域增加模糊或其他效果
    <style>
  .box {
    filter: blur(10px);
  }
</style>
<div class="box">
  <img src="../../img/animal1.jpeg" alt="" />
</div>

    <style>
  .box {
    display: inline-block;
  }
  .box img {
    display: block;
  }
  .box .cover {
    position: absolute;
    left: 0;
    right: 0;
    top: 0;
    bottom: 0;
    background-color: rgba(0, 0, 0, 0.5);
    backdrop-filter: blur(10px);
  }
</style>
<div class="box">
  <img src="../../img/animal1.jpeg" alt="" />
  <div class="cover"></div>
</div>
